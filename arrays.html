Arrays

Average of numbers using arrays:

				
class introduction
{
    public static void main(String args[])
    {
        int array[];
        array= new int[11];
        int i;
        int x=0;
        for(i=0;i<11;i++)
        {
            array[i]=x;
            x++;
        }
        float result=0;
        int j;
        for(j=0;j<11;j++)
        {
            result= result + array[j];
        }
        System.out.println("Average is: "+result/11);
    }
}

Declaration of array:

int array[]=new int[11]
is same as
int array[]; array= new int[11];
